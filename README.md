# SmartHealth AWS Infrastructure Automation

This repository contains the Infrastructure as Code (IaC) for a secure, highly available healthcare system migration to AWS.

## Key Features
* **VPC Architecture:** Multi-AZ deployment with public/private subnets.
* **Auto Scaling:** EC2 Auto Scaling group with Application Load Balancer.
* **Security:** Integrated AWS WAF and IAM roles for least-privilege access.
* **Storage:** S3 bucket with automated Lifecycle policies for cost optimization.
* **Serverless:** AWS Lambda function for automated security scanning.
